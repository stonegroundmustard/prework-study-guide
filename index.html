HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML attributes define addiction information about elements, and are always defined in the beginning, never the closing.
  • Declerations in HTML code are not casesensitive, such as the type.
  • The title of an HTML webpage is important for SEO's or Search Engine Optimizations: the process of improving the ranking of a website in a search engine.
  • There are many types of code language used in HTML coding including phrases to connect external resources to HTML documents, define metadata, make types of lists, input image information,etc.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git "feature": prefix to /"branch-name" for indication
  • git "add": modification in current branch
  • git "pull": is the command to receive branch modifications into local environment
  • git "push": is the command to send changes made locally to remote github branch

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • An array is a single variable used to hold a group of data; arrays can hold different types of data.
  • The first item in an array has an index(location) of zero.
  • "if" and "for" loops in JS are ways to manipulate the control flow by interupping it, or allowing a single code block to be executed repeatedly.
